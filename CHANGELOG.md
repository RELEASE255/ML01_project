## [1.0.0] - 2026-01-20
### Added
- Initial public release of ML01 DIY LED Board
- Complete hardware documentation and assembly guide
- Firmware with two operating modes (FULL and CHASE)
- Web interface using Microdot framework
- WiFi connectivity via Raspberry Pi Pico 2W
- NTP synchronization for CHASE mode
- 3D printable PCB stand models
- Bill of Materials (BOM)
- Technical schematics and drawings
- Multi-license structure (MIT, CC BY-SA, CC BY-NC-SA)
